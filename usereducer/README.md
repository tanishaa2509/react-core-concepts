# useReducer Demo

A simple React app demonstrating the `useReducer` hook with a counter example.

## What it does

Renders a counter with increment and decrement buttons, managed via `useReducer` instead of `useState`.

## Stack

- React 19
- Vite

## Getting started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
